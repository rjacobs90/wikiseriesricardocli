============
Installation
============

At the command line::

    $ pip install wikiseriesricardocli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv wikiseriesricardocli
    $ pip install wikiseriesricardocli

Or, if you are using pipenv::

    $ pipenv install wikiseriesricardocli
Or, if you are using pipx::

    $ pipx install wikiseriesricardocli
